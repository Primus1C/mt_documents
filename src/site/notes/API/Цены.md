# POST `/api/v1/orders`

**Название:** установка цен

**Описание:** cоздание документа [[📄Установка цен (продажи)\|📄Установка цен (продажи)]].

> [!important] 
> Документ [[📄Установка цен (продажи)\|📄Установка цен (продажи)]] задает цены на товар с начала суток даты самого документа. 

**Запрос**
    POST /trade/hs/stock/prices
    
**Параметры:** 
    warehouseId | warehouseCode | warehouse - обязательный (числовой код склада, или его наименование)
    priceType - тип цены
    currency - валюта
    items - массив товаров с ценами:
		itemCode - числовой код товара
		price - массив цен и порогов цены: priceQty, qty

**Пример запроса:**
```json
{
    "warehouse": "Склад Первый",
    "priceType": "Крупный опт",
    "currency": "Руб",
    "items": [
        {
        "itemCode": "3825",
        "price": [
                {"priceQty": 8000, "qty": 0},
                {"priceQty": 7700, "qty": 5},
                {"priceQty": 7400, "qty": 10},
                {"priceQty": 7100, "qty": 15}
            ]
        },
        {
        "itemCode": "3826",
        "price": [
                {"priceQty": 8002, "qty": 0},
                {"priceQty": 7702, "qty": 5},
                {"priceQty": 7402, "qty": 10},
                {"priceQty": 7102, "qty": 15}
            ]
        }
    ]
}
```
**Ответ**
```json
{
    "success": true,
    "error": "",
    "documentNumber": "000000013",
    "items": 2
}
```

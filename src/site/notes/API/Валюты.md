# GET `/api/v1/orders`

**Название:** получение валют
**Описание:** метод возвращает массив всех валют учетной системы 1С.

**Запрос**

    GET /trade/hs/stock/currencies

**Ответ**

```json
[
  {
    "currency": "USD",
    "currencyCode": "840",
    "sign": "$"
  },
  {
    "currency": "RUB",
    "currencyCode": "643",
    "sign": "₽"
  }
]
```

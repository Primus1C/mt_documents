# GET `/api/v1/delivery-notes`

**Название:** получение расходной накладной
**Описание:** получение расходной накладной в формате PDF или Excel

**Запрос**

    GET /trade/hs/stock/delivery-notes

**Параметры**

| параметр                                       | значения                    | описание                                                                                              | обязательный |
| ---------------------------------------------- | --------------------------- | ----------------------------------------------------------------------------------------------------- | ------------ |
| number \| deliveryNumber \| deliveryNoteNumber |                             | номер расходной накладной в 1С                                                                        | +            |
| date \| deliveryDate                           |                             | дата расходной накладной                                                                              | -            |
| format                                         | pdf \| excel \| xlsx \| xls | формат накладной (по умолчанию **pdf**)                                                               | -            |
| serials                                        | with \| without             | вариант накладной: с разделением по сериям, либо сводно по товарам без серий  (по умолчанию **with**) | -            |

**Ответ**


# POST `/api/v1/delivery`

**Описание:** запись расходной накладной с серийными номерами 
Количество товара каждой серии устанавливается равным 1.

**Запрос**

    POST /trade/hs/stock/delivery

**Параметры**

| параметр          | описание                              | обязательный |
| ----------------- | ------------------------------------- | ------------ |
| orderNo \| number | входящий номер заказа                 | +            |
| item \| items     | числовой код товара, или массив кодов | +            |

# GET `/api/v1/series`

**Описание:** метод возвращает серии товаров, допустимые для заполнения [[API/Документы отгрузки\|Расходной накладной]] указанного заказа (из имеющихся на остатках [[API/Склады\|склада]], указанного в [[API/Заказы\|Заказе]]) 

**Запрос**

    GET /trade/hs/stock/series

| параметр                                 | описание                                    | обязательный                                   |
| ---------------------------------------- | ------------------------------------------- | ---------------------------------------------- |
| incomingNo \| incomingNumber \| incoming | номер входящий (нумерация заказчика) Заказа | + (необязателен, если указан *номер 1С*)       |
| orderNo \| orderNumber \| order          | номер 1С                                    | + (необязателен, если указан *номер входящий*) |

**Ответ**

```json
	{
	"orderNumber": "00001234",
	"incomingNumber": "PO-12345",
	"items": [
		{
			"itemCode": "5678",
			"serials": ["SN001", "SN002", "SN003"]
		},
		{
			"itemCode": "4570",
			"serials": ["SN500", "SN501"]
		}
	]
}
```

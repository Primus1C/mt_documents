# POST `/api/v1/return`

**Описание:** Метод создаёт документ возврата по [[API/Документы отгрузки\|Расходной накладной]] и заполняет ее товарами по указанным серийным номерам (IMEI). 
Вместо реквизитов Расходной накладной можно указать реквизиты [[API/Заказы\|Заказа]].

**Запрос**

    POST /trade/hs/stock/return

**Параметры:**

| параметр                                                                           | описание                                                              | обзательный                                                |
| ---------------------------------------------------------------------------------- | --------------------------------------------------------------------- | ---------------------------------------------------------- |
| incomingNo \| incomingNumber \| incoming<br>или<br>orderNumber \| orderNo \| order | номер Заказа входящий (нумерация заказчика)<br>или<br>номер Заказа 1С | + (необязательно, если указан *номер расходной накладной*) |
| delivery \| deliveryNumber \| deliveryNo                                           | номер Расходной накладной                                             | + (необязательно, если указан *номер заказа*)              |
| employeeCode \| employeePin                                                        | код сотрудника 1С, или его пин                                        | +                                                          |
| serials                                                                            | массив серийных номеров                                               | +                                                          |

**Пример запроса:**

```json
{
    "incomingNumber": "API-0122",
    "serials": [
        5423467471235446,
        45678567867457,
        23452345674575678
    ]
}
```

**Ответ 200:**
```json
{
    "success": true,
    "error": "",
    "returnNumber": "МТ-010544",
    "deliveryNumber": "МТ-0356879",
    "orderNumber": "МТ-20260701-04",
    "incomingNumber": "API-0122",
    "serials": [
        "5423467471235446",
        "45678567867457",
        "23452345674575678"
    ]
}
```

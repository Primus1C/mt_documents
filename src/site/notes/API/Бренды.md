# GET `/api/v1/brands`

**Описание:** метод возвращает брэнды товаров (группы верхнего уровня справочника Номенклатура).

**Запрос**
    GET /trade/hs/stock/brands
    
**Параметры:** нет

**Ответ 200**
```json
[
    {
        "code": "15497",
        "brand": "Acer"
    },
    {
        "code": "13318",
        "brand": "AGM"
    },
    {
        "code": "22839",
        "brand": "AIBI"
    }
]
```

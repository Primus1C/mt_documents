# GET `/api/v1/employees`

**Описание:** 
метод возвращает сотрудников, собирающих товары по Заказам в Расходные накладные .

**Запрос**
    GET /trade/hs/stock/employees

**Параметры**

| параметр      | описание                                                                                                   | обязательный                                     |
| ------------- | ---------------------------------------------------------------------------------------------------------- | ------------------------------------------------ |
| search        | наименование заказчика (поиск происходит по вхождению этого параметра запроса в наименования контрагентов) | необязателен, если указан *код заказчика*        |
| warehouseCode | код склада                                                                                                 | необязателен, если указано *наименование склада* |
| warehouse     | наименование склада                                                                                        | необязателен, если указан *код склада*           |
**Ответ**

```json
{
    "warehouse": "Склад резерва",
    "warehouseCode": "2",
    "employees": [
        {
            "employee": "Полуэктов В.",
            "employeeCode": "11",
            "pin": "1234"
        },
        {
            "employee": "James Johnes",
            "employeeCode": "32",
            "pin": "5678"
        }
    ]
}
```

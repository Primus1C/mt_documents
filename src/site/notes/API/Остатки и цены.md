# GET `/api/v1/balances`

**Название:** 
    получение остатков и цен
**Описание:** 
    получение остатков и цен
**Запрос**
    GET /trade/hs/stock/balances
**Параметры:** 
    warehouseId | warehouseCode | code — код [[API/Склады\|склада]]
    priceType — [[Виды цен\|вид цены]]
    itemCode | itemCodes — код товара, или коды, разделенные запятой — необязательный параметр
**Примеры:** 
    /hs/stock/balances?priceType=Крупный опт&warehouse=Склад Дубай&itemCode=3825,3827

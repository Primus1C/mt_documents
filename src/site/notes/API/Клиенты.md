# GET `/api/v1/counterparties`

**Название:** 
    получение клиентов
**Описание:** 
    получение контрагентов заказов.
**Запрос**
    GET /trade/hs/stock/counterparties
**Параметры:** 
    search - поиск клиента по названию
    counterpartyCode | code - поиск клиента по коду
**Примеры:** 
    GET /trade/hs/stock/counterparties?page=1&pageSize=50
    GET /trade/hs/stock/counterparties?search=покупатель

# POST `/api/v1/counterparties`

**Название:**  
    создание клиентов
**Описание:**  
    создание контрагентов заказов
**Запрос**
    POST /trade/hs/stock/counterparties
**Параметры:** 
    counterparty | name | customer - обязательный
    email
    phone | phones
    city - город (заполняется только из уже существующих в 1С)
    address
    note | comment
    priceType  - [[Виды цен\|тип цены]], строка, строго один из вариантов 
	    Если вид цены не задан при создании клиента, регистрация его заказов (через API) будет невозможна, пока вид цены не будет установлен в 1С (метода API для изменения контрагентов пока нет)
    
**Пример запроса:**
```http
POST /trade/hs/stock/counterparties
Content-Type: application/json
{
  "counterparty": "Иванов Полуэкт",
  "email": "buyer@example.com",
  "phone": "+7 999 000-00-00",
  "city": "Москва",
  "address": "ул. Примерная, 1"
  "priceType": "Крупный опт"
}
```

**Ответ**
```json
{
    "success": true,
    "error": "",
    "counterparty": "Иванов Полуэкт",
    "counterpartyCode": "5323"
}
```

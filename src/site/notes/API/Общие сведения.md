### Метаданные

- **Авторизация:** `API-key` (требуются права: `client:write`)
    
- **Content-Type:** `application/json`
    
~~- **Идемпотентность:** Да, поддерживается через заголовок `X-Idempotency-Key`~~
### Формат данных

    — запросы с телом: JSON, заголовок Content-Type: application/json

    — ответы: JSON (кроме [[API/Документы отгрузки\|delivery-notes]] — бинарный файл)

    — коды справочников (itemCode, warehouseCode, counterpartyCode) — строки: "4", "42"

    — даты в ответах: yyyy-MM-ddTHH:mm:ss

    — даты в query/body: yyyy-MM-dd (из ISO-строки берутся первые 10 символов)

### Пагинация

Используется в [[API/Остатки и цены\|balances]], [[API/Клиенты\|counterparties]], [[API/Заказы\|orders]] (режим списка).  
  
В запросе отправляем :  

    page  — номер страницы, целое ≥ 1, по умолчанию 1

    pageSize  — размер страницы, 1–500, по умолчанию 500
    
В ответе всегда :

    "page"  — номер страницы

    "pageSize"  — размер страницы

    "totalItems" – общее количество объектов     
    
    "totalPages" – общее количество страниц
   
### КОДЫ ОТВЕТОВ И ОШИБКИ
   
   200  — успешный запрос
   400  — ошибка валидации, бизнес-правило, неверные параметры
   404  — заказ или накладная не найдены
   409  — попытка изменить отгруженный заказ
   500  — внутренняя ошибка сервера
### СВОДНЫЙ СПИСОК МЕТОДОВ

GET   /balances              — остатки товаров с ценами
GET   /warehouses            — список складов
GET   /currencies            — список валют
GET   /counterparties        — список контрагентов
POST   /counterparties        — создание контрагента
POST   /orders                — создание заказа
GET   /orders                — получение заказа
PATCH   /orders                — полная замена инвойса
PUT   /orders                — добавление позиций
DELETE   /orders                — удаление позиций / заказа
GET   /delivery-notes        — PDF/Excel расходной накладной
POST   /delivery       — создание расходной накладной из заказа
GET   /series                — получение серий, допустимых к отгрузке по заказу

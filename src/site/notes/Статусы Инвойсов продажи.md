### Допустимые значения при смене статуса (из статуса - в статус)

| в статус →<br>из статуса ↓       | зарегистрирован | изменен | собирается <br>(заблокирован) | отгружен | отменен |
| -------------------------------- | :-------------: | :-----: | :---------------------------: | :------: | :-----: |
| **<без статуса>**                |        +        |    -    |               -               |    -     |    -    |
| **зарегистрирован**              |        Х        |    +    |               +               |    -     |    +    |
| **изменен**                      |        -        |    Х    |               +               |    -     |    +    |
| **собирается<br>(заблокирован)** |        -        |    +    |               Х               |    +     |    +    |
| **отгружен**                     |        -        |    -    |               +               |    Х     |    -    |
| **отменен**                      |        -        |    +    |               -               |    -     |    Х    |

### Автоматическая установка статусов
- **Зарегистрирован** 
  - при проведении [[📄Инвойс продажи\|Инвойса продажи]], если *не Отгружен*
  - при отмене проведения расходной накладной, созданной на основании текущего  [[📄Инвойс продажи\|инвойса продажи]]
- **Изменен** - при сохранении измененного [[📄Инвойс продажи\|инвойса продажи]]
- **Собирается** (заблокирован для изменений)- при записи новой расходной накладной, созданной на основании текущего  [[📄Инвойс продажи\|инвойса продажи]]
- **Отгружен** - при проведении расходной накладной, созданной на основании текущего  [[📄Инвойс продажи\|инвойса продажи]]
- **Отменен** - при отмене проведения или пометке на удалении текущего  [[📄Инвойс продажи\|инвойса продажи]]
